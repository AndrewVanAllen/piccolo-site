Piccolo

What is Piccolo?

Piccolo is a lightweight CSS framework meant to give you a jumpstart in your development. Piccolo is not meant to be an all encompasing CSS framework, such as Bootstrap. If you want a framework with just the basics, Piccolo is for you.

Lightweight

Light as a leaf.

Simple

Just the basics.

Responsive

Changes with the times (or device).

Clean

Crisp as a glass of lemonade.

Documentation

Columns

Piccolo uses a standard column setup similar to other CSS frameworks. There are 12 columns in the grid that automatically stretch to fit the browser.

nine

three

two

ten

half

half

nine (offset three)

To create a column, simply use the column class. Example (from above): 


 <div class="row">
   <div class="column nine example"><p>nine</p></div>
   <div class="column three example"><p>three</p></div>
 </div>
 <div class="row">
   <div class="column two example"><p>two</p></div>
   <div class="column ten example"><p>ten</p></div>
 </div>
 <div class="row">
   <div class="column half example"><p>half</p></div>
   <div class="column half example"><p>half</p></div>
 </div>
 <div class="row">
   <div class="column nine offset-three example"><p>nine (offset three)</p></div>
 </div>

If you need more help, just refer to piccolo.css
